Boards

Boards contain columns and cards and are built up as the user adds these elements.
When ToDoo! starts, you will be presented with a blank board containing no columns or cards.


Saving boards

You can save your board to be able to load it up later and continue where you left off.
To save your board go to File > Save Board.
Your board will be saved to a file called "board.bin" in the application directory.
To save multiple boards you must copy or move the "board.bin" file to another location as this file will be overwritten each time a board is saved.


Loading boards

To load up an existing board, go to File > Load Board.
The file "board.bin" in the application directory will be loaded and the existing board will be displayed in the main window.
To load an existing board from another location, copy/move the .bin file into the application directory and rename it to "board.bin". Select File > Load Board and the existing board will be displayed.

Columns

A board can contain up to four columns.
Columns contain a heading, which can be customized, and you can also change the background colour with a choice of four preset colours to help define your board.
Columns will resize automatically as they are added and will display equally on the board.


Adding columns

To add a column, press the "Add Column" button in the lower, left-hand corner of the window.
A maximum of four columns can be added to a board.


Removing columns

To remove a column, press the "X" button located in the top right-hand corner of the column.
The remaining columns will automatically resize to match the board.


Removing all columns

To remove all columns, go to Edit > Clear Board.
This will remove all columns AND cards from the board, essentially creating a new board.


Changing a columns colour

To change the background colour of a column, press the "Colour" button located in the top right-hand corner of the column.
The background colour of the column will cycle through four preset colours to help you define your board.

Cards

A board can contain up to forty (40) cards simultaneously.
Cards contain a task heading and a task description field which can be modified.
Cards can be repositioned around the board by clicking and dragging.
Cards will resize based on the number of columns on the board to ensure it doesn't get too cluttered.


Adding cards

To add cards, press the "Add Card" button in the lower, left-hand corner of the window.
A maximum of forty cards can be on a board simultaneously. To add more you must first remove other cards.


Removing cards

To remove a card, press the "X" button located in the top right-hand corner of the card.


Removing all cards

To remove all cards from the board, go to Edit > Clear Cards.
All cards will be cleared from the board.
